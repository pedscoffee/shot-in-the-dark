# SmartChart

A stateless, local-first PWA for rapidly inserting reusable markdown templates into EMR notes.

## Features

- Auto-template matching
- Auto-copy to clipboard
- Offline support
- Floating draggable pane
- Markdown preview
- No backend
- HIPAA-conscious architecture

## Deployment

1. Upload files to GitHub repository.
2. Enable GitHub Pages.
3. Open deployed URL.
4. Install as PWA if desired.

## Security Model

- No PHI transmission
- No analytics
- No backend
- Fully local browser storage

## Development

Open index.html locally or use a static server.
